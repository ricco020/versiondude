HTML5 Parser (htmlparser)

The htmlparser project is an HTML5 parsing engine written in Java that provides SAX and DOM interfaces. Early development repositories were hosted here under /whattf/htmlparser/trunk/. The parser became the engine behind the Nu HTML Checker (validator.nu).
